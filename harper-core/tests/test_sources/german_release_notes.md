# Produktbericht für die Frühlingsversion

## Zusammenfassung

Die neue Version startet schneller und reagiert bei langen Listen deutlich ruhiger.
Ein zusätzlicher Datenbankserver verteilt Lastspitzen früher und hält die Oberfläche auch unter hoher Auslastung stabil.

## Verbesserungen

Die Fehlerbehebung im Importdialog verhindert doppelte Einträge.
Die Systemvoraussetzungen sind unverändert, aber der Speicherplatz wird effizienter genutzt.
Ein kurzer Versionshinweis erklärt die wichtigsten Änderungen für das Supportteam.
